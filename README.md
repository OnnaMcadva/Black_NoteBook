# Notebook Store

A simple Java project simulating a notebook store.  
Users can view all available notebooks or filter them based on RAM, screen size, and operating system.

## Features
- View all notebooks in stock
- Filter notebooks by OS, RAM, and screen size
- Graphical interface using `JOptionPane`

## How to run
1. Make sure you have Java installed.
2. Place all `.java` files in the same folder.
3. Place images in the `images` folder.
4. Compile all files:

```bash
javac *.java
